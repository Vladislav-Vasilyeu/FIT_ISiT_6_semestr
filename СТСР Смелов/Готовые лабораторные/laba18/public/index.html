Ответы на вопросы

  1. ORM - технология, которая отображает таблицы базы данных на объекты языка программирования и позволяет выполнять операции с данными через модели.
  2. Для работы Sequelize с Microsoft SQL Server нужно установить дополнительный драйвер tedious.
  3. Sequelize-схема - описание модели: имя таблицы, набор полей, типы данных, ключи, ограничения и связи с другими моделями.
  4. SELECT-запрос выполняют методы findAll, findOne, findByPk или raw-запрос через sequelize.query.
  5. INSERT-запрос выполняет метод create.
  6. UPDATE-запрос выполняют методы update или instance.update.
  7. DELETE-запрос выполняют методы destroy или instance.destroy.
  8. sequelizeSchema.hasMany задает связь «один ко многим»: одна запись исходной модели может иметь много связанных записей другой модели.
  9. Raw query - SQL-запрос, написанный вручную и выполненный через Sequelize без построения запроса методами модели.
  10. Code first - подход, при котором структура базы данных описывается в коде приложения, а затем по этому описанию создается или изменяется БД.
  11. Транзакция - группа операций с базой данных, которая выполняется как единое целое: все операции фиксируются вместе или вместе отменяются.
