# Лабораторная 11 (WebSocket)

Все приложения слушают/подключаются к `ws://localhost:4000`. Запускайте задания по отдельности (одновременно несколько серверов на одном порту не запустятся).

## Установка

```bash
npm i
```

## Задание 01 (прием файла, upload)

Терминал 1:

```bash
npm run 11-01
```

Терминал 2 (опционально путь к файлу):

```bash
npm run 11-01a -- .\somefile.txt
```

Файл будет сохранен в `upload`.

## Задание 02 (отдача файла, download)

Терминал 1:

```bash
npm run 11-02
```

Терминал 2:

```bash
npm run 11-02a
```

Клиент сохранит файл в `downloaded`.

## Задание 03 (broadcast + ping/pong)

Терминал 1:

```bash
npm run 11-03
```

Терминал 2/3/4 (несколько экземпляров):

```bash
npm run 11-03a
```

## Задание 04 (JSON запрос/ответ)

Терминал 1:

```bash
npm run 11-04
```

Терминал 2/3 (разные имена клиентов):

```bash
npm run 11-04a -- Alice
npm run 11-04a -- Bob
```

## Задание 05 (RPC, rpc-websockets)

Терминал 1:

```bash
npm run 11-05
```

Терминал 2:

```bash
npm run 11-05a
```

Параллельно:

```bash
npm run 11-05b
```

Вычисление выражения:

```bash
npm run 11-05c
```

## Задание 06 (publisher/subscriber)

Терминал 1:

```bash
npm run 11-06
```

Терминал 2/3/4:

```bash

npm run 11-06b
npm run 11-06c
```

В терминале сервера вводите `A`, `B` или `C`.
